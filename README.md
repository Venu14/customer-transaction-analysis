# Customer Transaction Analysis 🚀

##  Overview
Simulated a real-world transaction dataset (5000+ records) and analyzed fraud patterns and customer behavior using Python and SQL.

## Tools Used
- Python (Pandas)
- SQL

## Key Analysis
- Fraud detection
- High-value transactions
- Location-based revenue
- Customer spending patterns

## Key Insights
- Fraud transactions identified using patterns
- High-value transactions tracked
- Revenue distribution across cities analyzed

## Business Impact
Helps detect fraudulent activities and improve financial decision-making.
